Site Feed "Stand By" Announcement

When it is necessary to introduce a standby site feed, one with the following message is employed:


We Are Experiencing Technical Difficulties
 
Please Stand By
 
Incident Response Procedures are in effect.
This site feed is suspended to prevent access to corrupted data.
 
Updated status will be posted as the situation becomes clear.
 
Meanwhile, there may be damaged or missing pages.
 Follow links in feed entries with extreme caution.
 

This will be the only entry in the feed, which is prepared in advance and stored in the standard feed location whenever it is necessary to interpose it in place of a (possibly-) corrupt site feed.

This message assumes that status for this site can be found as part of the combined status reported on Orcmid's Lair.

Standby Feed XML

The file is named pn-atom.standby.xml.  It is stored in the same directory as as pn-atom.xml so that it can be quickly swapped in place of the feed.

After any exercise of the file, it needs to be prepared for the next usage.  This is accomplished by 

  • Advancing the <modified> element for the standby feed itself

  • Advancing the <modified> and <issued> elements of the single entry

  • Advancing the number in the <title> and the <id> of the single entry.

When the feed is being used in a test, it is a simple matter to take the current copy of the standby feed and update its <modified> and <issued> timestamps to be later than the time of the test entry and earlier than the intended time of the all-clear entry.
