  • This page:  FrontPage Document Engineering 0.1.0 <https://orcmid.github.io/miser/construction/templates/00/c000006c.htm>.  Provides the Include Page and the procedure for its usage via Microsoft Office FrontPage 2003 for the Miser Project on GitHub..

Using Microsoft Office FrontPage 2003, the correct ClustrMap images may be inserted into any number of site pages.  Insertion is by using the FrontPage extensions "Include Page" web component.  The Include Page provides a single-cell table element that accomplishes four purposes:

  1. Browser visits to the page are counted as a visit to an Orcmid on GitHub page.

  2. A miniature image showing the geography and varietyof visitor locations is presented in the cell.

  3. The image provides a link to a larger map and detailed information about the statistics and the ClustrMaps service.

  4. Maintenance of the insertion is performed in a single place and automatically reflected in all of the places the ClustrMap has been inserted:

The link in the caption beneath the cell provides access to this folio and its explanation.  The propagation of Include Page changes to the individual pages is accomplished at design time in the GitHub repository.  The pages transferred to the hosting site and served to browsers simply have the inclusion as if it is a direct part of the HTML page.

Note that the image does not appear except where the reference is from the registered web site as the referrer.

1. Insert Usage

The Include Page should be inserted exactly once on any page that is intended to be included in the page visits statistic and mapping.  The current version of the Include Page is always available at web site page construction/templates/00/c000006b1.htm.

The Include Page introduces a single-celled table, with caption.  By including this cell in a standalone paragraph or an existing paragraph, it can be positioned and aligned using FrontPage formatting features.

For inclusion in Markdown pages, the normal positioning is by copying the HTML code directly from c000006b1.htm into a Markdown table cell.

2. Current Tracked-Visit Locations 

In FrontPage, the hyperlink view of the Include Page will show all of those pages that refer to it, either as a web component or by direct linking.

3. Historical ClustrMaps

The ClustrMap accumulates visits and, at some point, will be reset to start anew.  Successive accumulations may reveal differences in the location of interest as well as differences in the rate at which visits occur.
