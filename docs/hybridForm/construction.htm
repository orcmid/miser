This is the construction zone for the hybridForm Document-Format Exemplar specialized to the Miser Project of Orcmid on GitHub.  This is an exemplar of boilerplate for introduction of technical topics as peers of this topic.  It's Construction Structure and Zones constitute exemplars for the necessary customization of portions to be replicated in the organization of topical content.  This paragraph should be updated as part of customization in a topic/ folder that is primed with a copy of hybridForm/.

It is important to understand that this Construction Zone serves as both the construction structure for the hybridForm topic, and is itself a customizable exemplar for other topics at the same level as hybridForm/ in the project-repository documentation where this folder has been created.

Exemplar Customization

This exemplar is designed for automatic incorporation of components using Microsoft Office FrontPage 2003 provisions.  Editing by other means is not advisable.

hybridForm is a template and exemplar for repurposing as miser project web topic sections with their own article subsections.  This current Article Diary & Job Jar page is the exemplar for topic/ Construction Zones.  It is situated in hybridForm/ as a peer level of a chosen topic/ to be customized completely or partially with material from this hybridForm/ folder.

The practice is to have the topic/c000000.htm and construction.htm always provide a Construction Zone for the introduced topic/.  This Construction Zone's c000000.htm exemplar provides initial checklists of work items and additional ones that arise in establishment of a customized topic/ subdirectory of web pages.  If there is to be no such (preserved) Construction Zone, this material should be deleted and also removed from the immediate infrastructure on the topic/index.htm page after customization is completed.  The portable practice is to keep things familiar and always have the Construction Zone present.  Usage will require employment of an HTML editor.  Customization is as follows.

  1. In the top banner, the site/project construction image should already appear correct in the left cell.  The link should be to "../", the project home directory.
  2. In the title section of the top banner, the "hybridForm" phrase should be replaced entirely by the topic name. 
  3. The Page Properties title should also be adjusted in the same manner.
  4. In the far right block of the top banner, without altering the links, change the topic name from "hybridForm" to the name of the new topic directory..  The link should remain "./".  The "construction" link should be to "construction.htm".
  5. No changes should be required in the footer banner except for the "created" entry and possible identity and link to the author.  Verify that there are no mistaken links in that banner.
  6. In the "Local Zone" section of the table, replace "hybridForm" with "topic".  There should be no other changes in the table.  If there is to be no Construction Zone at this level, this page will not be used.
  7. The tracking of paving/repaving/repurposing notification in the status this Construction Zone's customization is complete.  If that is not aligned with the current situation, adjust to employ the correct notification, if any.
  8. When the customization is complete, delete this complete "Exemplar Customization" section and commit this repurposed version to the repository.
