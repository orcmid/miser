Exemplar Customization

  1. hybridForm is a template and exemplar for repurposing as MiserProject topic sections with their own article subsections.  This current Construction Structure page is repurposed for creation of an annual article subsection when copied into a folder at the same level under the project repository as is miser/hybridForm/yyyy/index.htm.  Customization is as follows.
  2. In the top banner, the construction structure image should already appear correct in the left cell.
  3. In the title section of the top banner, the phrase "hybridForm/yyyy" should be altered to the name of the topic and the year of the folder in which this page has been placed.
  4. In the far right block of the top banner, without altering the links, change the topic name from "hybridForm" to the topic folder being edited.  The link should remain "../".
  5. In the far right block of the top banner, without altering the links, change the annual folder name for "yyyy" to the year in which articles are being (or have been) initiated.
  6. In the Superior Structure section, replace both "hybridForm/" occurrences with the topic-folder name used in the modified title.
  7. If the Superior Structure location does not have a Construction Zone, replace with a link (and correctly named) Construction Zone of the nearest superior Construction Zone.  There will always be one at some level.
  8. For Local Infrastructure, adjust "hybridForm/yyyy" to the actual location path name.  It should still be a link to "index.htm".
  9. For Subordinate Structure, adjust "mm" to the actual month number in which one or more articles are initiated. Correct the link accordingly.  Replace "month, yyyy" with the correct date period (e.g., "February, 2024").  Note: As articles are initiated in additional months, additional Subordinate Structure entries will be made here.
  10. Adjust the Revision History to reflect the initiatialization of this page.  If 0.0.0 is retained, the cross-linking of version number there and in the rightmost title block will already be set properly.  The text might link to this page under hybridForm as the template source.
  11. If the superior Construction Zone is changed in Superior Structure, the link in the hard hat logo of the footer banner needs to be adjusted accordingly.
  12. No other changes should be required in the footer banner except for the "created" entry and possible identity and link to the author.  Verify that there are no mistaken links in that banner.
  13. When the customization is complete, delete this complete "Template Customization" section and commit this repurposed version to the repository..
